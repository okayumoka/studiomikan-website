製品についての注意事項:

3Dプリンター製品に関しては、あらかじめ以下の点についてご了承いただきますようお願いいたします。

  • 3Dプリンター製ヨーヨーの仕様として、プレイに支障のない程度のブレがあります。このブレを最小限に抑えるため、XYZ YoYo Studioのヨーヨーは締め込みができるようになっています。 ただし締め込み過ぎるとヨーヨー本体が破損する可能性があるため、十分注意して作業してください。
  • 3Dプリンター製品はその材質上、熱に弱くなっております。 約55℃を超えると熱による歪みが発生し、さらに高温になると製品自体が溶け出す可能性があります。保管、持ち運びの際は熱に十分ご注意ください。 特に夏の車内など、高温になる可能性のある場所には放置しないようご注意ください。
  • 3Dプリンター製品には、製造工程由来の積層痕があります。これは仕様となりますので、あらかじめご了承ください。 サンドペーパー等で研磨することで積層痕を消すことも可能ですが、保障外となりますのでご注意ください。
